Preguntas Técnicas

Preguntas que podrían realizar durante la defensa del proyecto.

¿Por qué SHA-256?

Porque permite detectar cualquier alteración en la evidencia mediante una huella digital única.

¿Por qué usar RSA?

Porque garantiza autenticidad y no repudio mediante criptografía asimétrica.

¿Cómo detecta alteraciones?

El sistema recalcula el hash SHA-256 y lo compara con el hash original almacenado.

¿Cómo garantiza cadena de custodia?

Registrando automáticamente usuario, fecha, IP y acción realizada sobre cada evidencia.
